# 📑 Paper Translator Skill

这是一个设计给 IDE AI Coding Agent 的学术论文高保真全量解析与精译助手配置库。

## 📁 仓库内容

- **`SKILL.md`**：定义了学术论文精读与术语库沉淀系统的执行规则与 Agent 提示词配置。

## 🛠️ 执行规范

本配置包含以下三种执行模式：
1. **模式 A：一键初始化开荒模式**
   - 触发条件：`INIT: [论文唯一标识_年份]`
2. **模式 B：全量解析模式**
   - 触发条件：`PARSE: [论文唯一标识_年份]` 后接论文原文
3. **模式 B-Lite：精简解析模式**
   - 触发条件：`PARSE_LITE: [论文唯一标识_年份]` 后接论文原文
